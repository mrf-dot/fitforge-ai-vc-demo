# OpenClaw Workout App

VC Pitch-Ready Prototype for an AI-powered fitness assistant.

## Overview
A runnable prototype showcasing an AI-powered workout generator with a Flask backend, a lightweight React-like UI (static HTML in this version), and an OpenRouter key integration for future AI features.

## Problem
People struggle to create effective, personalized workouts without guidance or accountability.

## Solution
An end-to-end prototype that takes user goals, fitness level, and equipment and returns a personalized workout plan. Designed to be expanded with real AI, user accounts, and persistence.

## Tech Stack
- Backend: Python, Flask
- AI: OpenRouter (to be integrated via OPENROUTER_API_KEY in .env)
- Frontend: HTML/CSS/Vanilla JS
- DevOps: GitHub hosting, CI for MVP

## How to Run
1. Ensure dependencies: python3, pip, Flask, python-dotenv
2. Run: python3 app.py
3. Access: http://localhost:8080
